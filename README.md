# STM32WL3x-Demo-Workshop
In this workshop, ST experts introduce STM32WL3x RF , the latest Sub-1Ghz RF SoC offering the richest functionalities in the series.


This repository integrates the full PDF material used during [STM32WL33 Demo Workshop event](https://content.st.com/stm32wl3x-workshop-emea.html) 


1. Introduction 
2. Understand STM32WL3x through product presentation.
3. STM32WL3x unboxing and build your first RF application from STM32CubeMX
4. Demonstration of STM32WL3x radio flexibility (packet handler, modulation, interoperability)
5. STM32WL3x power consumption state-of-the-art (active, RF mode, standby) 
6. wM-Bus solutions for STM32WL3x
7. mioty solutions for STM32WL3x
8. RF Lab certification and support

In  repository you will also find the binaries used during above demonstrations.
